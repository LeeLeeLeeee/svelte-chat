# SVELTE-CHAT

### 소개
svelte로 구축한 프론트 환경과 go + echo로 구축한 API 서버간의 websocket 통신
redis를 pub-sub 기능을 활용하여 채팅방 토이 프로젝트 구축. 개발 환경 세팅은 docker 활용

### 기술 스택
#### front
- javascript
- svelte
- svelte-kit
- socket.io
#### back
- go
- echo (서버 구축)
- gorilla/websocket (소켓 통신)
- redigo (레디스 관리)
#### devops
- docker
#### test
- jest
#### design
- tailwind css
